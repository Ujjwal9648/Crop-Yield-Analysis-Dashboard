🌾 Smart Agriculture Analytics Dashboard

This project presents an interactive Power BI dashboard that analyzes agricultural data from sensors to harvest outcomes. 
The dashboard helps understand how environmental factors such as humidity, rainfall, soil moisture, fertilizer type, irrigation methods, and sunlight hours impact crop yield.

The goal of this project is to demonstrate how data analytics can support smart farming decisions and improve agricultural productivity.

This dashboard provides insights into:

• Crop yield based on disease conditions  
• Yield comparison by fertilizer type  
• Irrigation impact on crop productivity  
• Regional crop yield analysis  
• Seasonal trends in sowing and harvesting  
• Environmental factors affecting yield

📊 Key Insights

• Drip irrigation shows the highest crop yield compared to other irrigation methods.
• South India has the highest average crop yield among the regions analyzed.
• Crop diseases slightly reduce yield but proper fertilizer use can compensate.
• Harvest yield increases significantly between April and August.
• Sunlight hours and fertilizer combination strongly influence productivity.
